# simple_shell

A basic UNIX command line interpreter built for Holberton School.

## Features

- Executes simple shell commands using full path (e.g., `/bin/ls`)
- Handles Ctrl+D (EOF)
- Displays custom prompt `$ `

## Compilation

```bash
gcc -Wall -Werror -Wextra -pedantic -std=gnu89 *.c -o hsh

Usage
./hsh
